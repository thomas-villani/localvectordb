Need an API Key?

If you're running LocalVectorDB with authentication enabled, you'll need an API key to access the inspector interface.

To generate an API key:

lvdb auth create --name "Inspector Access" --expires 90

For development: You can disable authentication by setting require_api_key = false in your configuration.

🛡️ Security Note

API keys provide full access to your LocalVectorDB system. Keep them secure and never share them in public repositories or logs.
